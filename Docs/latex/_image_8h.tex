\doxysubsection{Detailed Description}
Provides an \doxylink{class_image}{Image} class for basic image processing operations. 

The \doxylink{class_image}{Image} class supports basic image processing functionalities including loading, saving, and converting images to grayscale. It is designed with a focus on handling image data efficiently while providing a simple interface for image manipulation. This class handles images as arrays of unsigned char, representing pixel data, and provides methods to load images from files, save images to files, and convert color images to grayscale. The class also explicitly bans copy construction and assignment to avoid unintentional deep copies of image data. It is a part of the utility toolkit developed by the Advanced Programming Group.
