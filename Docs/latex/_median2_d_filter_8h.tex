\doxysubsection{Detailed Description}
Implements a Median 2D filter for noise reduction in images. 

The \doxylink{class_median2_d_filter}{Median2\+DFilter} class applies a median filtering operation to 2D images to reduce noise while preserving edges. This technique is particularly useful in pre-\/processing for other image analysis tasks, as it helps to reduce salt-\/and-\/pepper noise without blurring sharp edges in the image. The filter supports customizable kernel sizes and padding types, allowing for flexibility in handling different types of images and noise levels. Part of the tools developed by the Advanced Programming Group, this class enhances the suite of image processing capabilities with a focus on maintaining image quality during noise reduction.
