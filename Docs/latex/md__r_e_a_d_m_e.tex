 This is an implementation for the {\bfseries{Advanced Programming Group Project}}.

This project provides a comprehensive toolkit for 2D/3D image processing, including utilities for loading, manipulating, and saving both 2D and 3D image data. It is designed to facilitate advanced image processing tasks such as filtering, projection, slicing of volumetric data, and edge detection. This library is particularly suited for applications in medical imaging, scientific visualization, 3D data analysis, and real-\/time image enhancement.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{\doxylink{class_image}{Image} Loading and Saving}}\+: Supports loading and saving images in various formats through the integration of the {\ttfamily stb\+\_\+image} and {\ttfamily stb\+\_\+image\+\_\+write} libraries.
\item {\bfseries{\doxylink{class_volume}{Volume} Manipulation}}\+: Allows the construction of a 3D volume from a series of 2D slices, along with the capability to save slices or projections of the volume back to disk.
\item {\bfseries{Filtering}}\+: Implements median, Gaussian, edge detection, and pixel-\/level filtering for noise reduction, smoothing, and edge enhancement of volumetric data.
\item {\bfseries{Projections}}\+: Offers functions for maximum, minimum, average, and median intensity projections, which are essential for visualizing structural information in volumetric data.
\item {\bfseries{Slicing}}\+: Provides functionality to extract arbitrary slices from a volume, facilitating the examination of cross-\/sectional data.
\item {\bfseries{Edge Detection}}\+: Incorporates edge detection algorithms such as Sobel and Prewitt, enabling the identification of edges within images for analysis and processing.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Project Structure}{Project Structure}}\label{md__r_e_a_d_m_e_autotoc_md2}
