\doxysubsection{Detailed Description}
Entry Point for the Unit Testing Framework. 

This file acts as the entry point for the comprehensive unit testing framework designed to verify the correctness and functionality of various image processing algorithms and utilities. It includes tests for classes such as \doxylink{class_test_algorithm}{Test\+Algorithm}, \doxylink{class_test_image}{Test\+Image}, \doxylink{class_test_projection}{Test\+Projection}, \doxylink{class_test_slice}{Test\+Slice}, \doxylink{class_test_volume}{Test\+Volume}, \doxylink{class_test_padding}{Test\+Padding}, \doxylink{class_test_pixel_filter}{Test\+Pixel\+Filter}, \doxylink{class_test_box2_d_filter}{Test\+Box2\+DFilter}, \doxylink{class_test_gaussian2_d_filter}{Test\+Gaussian2\+DFilter}, \doxylink{class_test_median2_d_filter}{Test\+Median2\+DFilter}, \doxylink{class_test_edge_filter}{Test\+Edge\+Filter}, \doxylink{class_test_gaussian3_d_filter}{Test\+Gaussian3\+DFilter}, \doxylink{class_test_median3_d_filter}{Test\+Median3\+DFilter}, and other related test classes. Each class targets specific functionalities within the image processing library, ensuring that operations such as filtering, projection, slicing, and volume manipulation work as expected. The STB \doxylink{class_image}{Image} library is utilized for image reading and writing operations, underlining the framework\textquotesingle{}s reliance on external libraries for handling image data. The main function orchestrates the initiation and execution of all unit tests, aggregating and presenting the results to provide a clear overview of the test outcomes.
