\doxysubsection{Detailed Description}
Defines the \doxylink{class_pixel_filter}{Pixel\+Filter} class for pixel-\/level image processing operations. 

\doxylink{class_pixel_filter}{Pixel\+Filter}, implementing the \doxylink{class_i_filter2_d}{IFilter2D} interface, offers a variety of pixel-\/level manipulations for 2D images, including grayscale conversion, brightness adjustment, histogram equalization, thresholding, and noise addition. It supports operations in different color spaces (HSL, HSV) and allows for configurable processing parameters such as brightness level, threshold for pixelation, and noise percentage. This class facilitates detailed image editing and enhancement tasks, making it a versatile tool in the Advanced Programming Group\textquotesingle{}s image processing toolkit. Whether for pre-\/processing steps or standalone adjustments, \doxylink{class_pixel_filter}{Pixel\+Filter} provides robust solutions for refining image aesthetics and quality.
