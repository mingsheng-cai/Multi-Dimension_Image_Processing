Tests the Generation of the Gaussian Kernel

Verifies the Gaussian2DFilter's ability to generate a Gaussian kernel with specified dimensions and sigma value. The kernel's sum must equal 1 to maintain image brightness, and its center value should be the maximum, reflecting the characteristics of a Gaussian distribution. This test ensures the kernel is constructed correctly, which is crucial for achieving the desired blurring effect.
