Detailed Description

Manages and manipulates 3D volumetric data.

The Volume class encapsulates 3D volumetric data, providing functionalities to load data from multiple image files or a directory, update internal data, and save specific slices or projections to files. It supports extracting slices along 'x-y', 'x-z', or 'y-z' planes and computing various intensity projections such as Maximum, Minimum, Average, and Median Intensity Projections. This functionality is essential for analyzing 3D data in medical imaging, scientific visualization, and other fields requiring volumetric data manipulation. The class ensures efficient handling of large datasets and integrates with external libraries like stb_image for reading and writing image files, making it a versatile tool for 3D data processing.
