Detailed Description

Defines the PixelFilter class for pixel-level image processing operations.

PixelFilter, implementing the IFilter2D interface, offers a variety of pixel-level manipulations for 2D images, including grayscale conversion, brightness adjustment, histogram equalization, thresholding, and noise addition. It supports operations in different color spaces (HSL, HSV) and allows for configurable processing parameters such as brightness level, threshold for pixelation, and noise percentage. This class facilitates detailed image editing and enhancement tasks, making it a versatile tool in the Advanced Programming Group's image processing toolkit. Whether for pre-processing steps or standalone adjustments, PixelFilter provides robust solutions for refining image aesthetics and quality.
