double TestBox2DFilter::calculateVariance ( const Image & image)
inlineprivate

Calculates the Variance of Pixel Values in an Image

Computes the variance of pixel values in an image to assess the distribution of pixel intensities. The variance provides a measure of the spread of pixel values around the mean, indicating the level of contrast and detail present in the image. This function calculates the variance by first computing the mean pixel value and then summing the squared differences between each pixel value and the mean. The final variance is obtained by dividing this sum by the total number of pixels.

Parameters
imageThe Image object for which to calculate the variance
Returns
The variance of pixel values in the image
