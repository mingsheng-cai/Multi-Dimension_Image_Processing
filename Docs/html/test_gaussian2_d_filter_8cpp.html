void test_Gaussian2DFilter ( )

Function to test the Gaussian2DFilter class for Gaussian filtering on an image.

This function creates an image with specified dimensions and data, applies a Gaussian2D filter with a given kernel size and standard deviation, and then compares the result with an expected output.

Note
This function assumes that the Image and Gaussian2DFilter classes are properly implemented.
